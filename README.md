# FocusSense – Study Behavior Analyzer

## What this project does

FocusSense is a simple Python tool that analyzes how effectively a student uses their time while studying.

Instead of just tracking time, it looks at patterns and highlights where focus is breaking.

## Problem

Students often believe they are studying enough, but distractions silently reduce actual productivity. There is usually no simple way to measure this behavior.

## Solution

This tool allows users to enter multiple study sessions and compares study time vs distraction time.

It then:
- Calculates overall productivity
- Identifies the weakest session
- Provides feedback based on behavior

## How to run

1. Make sure Python is installed
2. Clone the repository
